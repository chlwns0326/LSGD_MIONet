# LSGD_MIONet

Python (with JAX) implementation of the experiments of hybrid least squares/gradiet descent method for MIONets.

This repository contains data and python codes for numerical experiments of https://arxiv.org/abs/2607.06976

This is a subsequent research of hybrid least squares/gradiet descent method for DeepONets. (https://arxiv.org/abs/2508.15394, https://github.com/chlwns0326/LSGD_DeepONet)
